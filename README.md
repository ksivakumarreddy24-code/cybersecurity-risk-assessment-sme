# Cybersecurity Risk Assessment Framework for SMEs

**Minor Project 1 — Cybersecurity Internship (Naviotech Solution Pvt Ltd)**

## Objective
To develop a structured risk assessment model for evaluating cybersecurity risks in small and medium enterprises (SMEs), and to propose practical, cost-effective mitigation strategies to improve their security posture.

## Problem Statement
SMEs are frequent targets of cyberattacks due to limited budgets, minimal dedicated IT security staff, and low security awareness. This project builds a repeatable framework SMEs can use to identify, score, and mitigate their most pressing security risks.

## Methodology
- Researched common SME cybersecurity risks (phishing, ransomware, insider threats, weak passwords, unpatched systems, etc.)
- Built a risk assessment model aligned to the **NIST Cybersecurity Framework (CSF)** — Identify, Protect, Detect, Respond, Recover
- Designed a quantitative risk evaluation metric: **Risk Score = Likelihood × Impact** (1–5 scale each), mapped to Critical/High/Medium/Low priority bands
- Mapped each risk to specific, affordable mitigation strategies (MFA, firewalls, employee training, backups, etc.)
- Validated the model against two illustrative SME case studies (retail and professional services)

## Key Findings
- Phishing and ransomware consistently scored as the highest-priority risks across both case studies
- Low-cost controls (MFA, password managers, staff training, tested backups) meaningfully reduced risk exposure without requiring enterprise-level budgets
- A simple Likelihood × Impact scoring approach is usable by SME owners without a dedicated security background

## Tools & Concepts Used
- NIST Cybersecurity Framework (CSF)
- Risk scoring matrix (Likelihood × Impact)
- Case study validation methodology
